### 📂 프로젝트 목표
- 시계열 분석을 통해 스낵 시장의 반기별 매출을 예측하고 미래 시장 흐름을 전망
- 소비자 리뷰와 블로그 언급량 데이터를 활용해 감정 및 매출 간의 상관관계를 분석
- 트렌드 변화가 매출에 미치는 영향을 평가하고 구매 결정 요인을 도출
<br></br>
